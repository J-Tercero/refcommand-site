# Design system

## Colors

Charcoal `#1b1d1c` anchors the interface, warm off-white `#f6f0e4` provides the reading surface, and officiating gold `#dcae44` supplies emphasis and focus indicators.

## Typography

System sans-serif typography supports speed and legibility. Georgia is used for large editorial-style display headings. Body text uses a comfortable 1.6 line height.

## Spacing

The base responsive spacing token ranges from `1.25rem` to `2.5rem`. Sections use generous responsive vertical padding and cards use a minimum `1.5rem` inset.

## Cards and buttons

Cards use thin charcoal rules, warm surfaces, direct headings, and clear destination affordances. Primary buttons are gold with charcoal text. Text links retain visible underlines.

## Navigation

The header uses a semantic navigation landmark. On small screens it becomes an accessible toggle; without JavaScript the content remains available through direct URLs and the homepage.

## Breakpoints

Layouts begin mobile-first. Navigation and grids enhance at 700px, with wider refinements at 1040px.

## Accessibility expectations

Maintain semantic headings and landmarks, visible keyboard focus, sufficient contrast, no horizontal overflow, meaningful link labels, and reduced-motion support. Never rely on color alone to communicate status.
